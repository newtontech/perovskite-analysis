## 一、验证策略

### 1.1 三级验证框架

```
Level 1: 离线验证（交叉验证）
    ↓
Level 2: 在线验证（留出测试集）
    ↓
Level 3: 实验验证（实验室合成）
```

### 1.2 验证指标

| 指标 | 目标 | 验证方法 |
|------|------|----------|
| R² | ≥ 0.85 | 测试集评估 |
| MAPE | ≤ 8% | 测试集评估 |
| 外推误差 | ≤ 15% | 分布外数据 |
| 实验验证成功率 | ≥ 80% | 实验室合成 |

## 三、MLOps流程

### 3.1 持续训练

```yaml
# .github/workflows/train.yml
on:
  schedule:
    - cron: '0 0 * * 0'  # 每周日训练
  workflow_dispatch:

jobs:
  train:
    steps:
      - name: Fetch new data
      - name: Train model
      - name: Evaluate model
      - name: Deploy if better
```
